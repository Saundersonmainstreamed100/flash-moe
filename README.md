## 🛠️ Troubleshooting

### The app does not open
Try these steps:

- Right-click the app and choose Run as administrator
- Check that the file finished downloading
- Move the app to a simple folder like `C:\flash-moe\`
- Make sure Windows did not block the file
- Restart your PC and try again

### The model will not load
Try these steps:

- Use a smaller model
- Close other apps
- Check that you have enough RAM
- Make sure the model file is not damaged
- Put the model in a folder with a short path

### The app runs too slowly
Try these steps:

- Close browser tabs
- Plug in your laptop
- Use a smaller model
- Reduce any quality or precision setting
- Reboot and run only flash-moe

### Windows blocks the file
Try these steps:

- Right-click the file and open Properties
- Look for an Unblock option
- Apply the change if you see it
- Download the release again from the official release page

### The screen looks blank
Try these steps:

- Wait a minute for the app to finish loading
- Minimize and restore the window
- Check if the app opened on a second screen
- Restart the app
